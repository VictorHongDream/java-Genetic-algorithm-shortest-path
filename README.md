## java-routing
在网页点击然后服务器使用遗传算法计算后返回最短路径
使用POST的请求

### 基本说明
使用Socket来完成内容，不使用servlet和tomcat
仅使用Socket绑定8080端口进行通信
使用了D3可视化库进行画图。
使用原生ajax进行JavaScript前端和后台进行交互的交互。


### 源文件和用法
src文件中为Java的源文件，WebContent为网页内容的文件夹

使用方法:在bin文件夹下

`$ sudo java SimpleHttpServer`

如果想后台运行加一个&即可

`$ sudo java SimpleHttpServer`
